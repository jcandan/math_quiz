#Math_Quiz

A light-weight interactive math quiz cli.

##Getting Started

With this Programme you can create Math-Quizes start and end will be user defined

###Prerequisties

Python 3.*

###Installing

Clone or download the zip and Extract it
Run math_quiz.py using python 3.*
